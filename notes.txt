Component Convention Order
------------------------------
Refs: Reactive references declared using ref.
Computed Properties: Computed properties defined using computed.
Methods: Any methods that will be used within the component.
Lifecycle Hooks: Hooks like onMounted, onUnmounted, etc.
Watchers: Any watchers defined for reactive data 34.
